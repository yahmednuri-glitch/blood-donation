# Nalut Central Hospital – Blood Donation System

This web application is an official initiative to organize blood donors
for Nalut Central Hospital.

## Features
- Donor registration
- Blood type search
- Mobile and desktop support
- Fast and simple interface

## Usage
Open the website and:
1. Register donor information
2. Search for available blood types
3. Contact donors directly by phone

## Note
All data is stored locally in the user's browser.

Developed for Nalut Central Hospital.
